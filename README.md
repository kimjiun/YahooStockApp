# YahooStockApp  

android app that shows stock information with yahoo finance API  
if you want to use this code put your Yahoo Finance API key in com.kimjiun.yahoostockapp.yahoo.YahooService.java file

------------

### Features
    Yahoo Finance API
    RxJava  
    StorIO
